## Design System Used

### Color Palette
- **Background:** #f5f0e8 (warm cream)
- **Text:** #1a1614 (dark brown/black)
- **Muted:** #a09890 (warm gray)
- **Gold:** #b8956a (accent gold)
- **Cyan:** #06b6d4 (Wooting brand color - primary)
- **Violet:** #8b5cf6 (secondary accent)

### Typography
- **Headings:** Space Grotesk (600-700 weight)
- **Body:** Outfit (400-500 weight)
- **Monospace:** Fira Code (code/specs)

### Sample Data Used
- **Keyboard:** Wooting 60HE+
- **Brand:** Wooting (cyan color)
- **Weight:** 360g
- **Switch Type:** Lekker Hall Effect
- **Polling:** 8000Hz
- **Layout:** 60%
- **Keycaps:** PBT Doubleshot
- **Connection:** Wired USB-C
- **Rating:** 9.4 / 10.0
- **Pro Usage:** 15.2%
- **Price:** $175
- **Pro Players:** s1mple, TenZ, donk, ZywOo, NiKo (5 total)
- **Related:** Razer Huntsman V3 Pro, Logitech G Pro X TKL

## Implementation Notes

1. **Self-contained:** Each HTML file includes all CSS inline. No external stylesheets needed.
2. **Responsive:** All designs include basic mobile/tablet responsiveness
3. **Production-ready:** Clean, semantic HTML with proper font loading
4. **Accessible:** Proper heading hierarchy, alt text structure, semantic elements
5. **Warm Palette:** All designs use the specified warm neutral palette with cyan accents
